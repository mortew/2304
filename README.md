# 2304

Скрипт для анализа гаплогрупп Y-хромосомы с помощью yleaf (https://github.com/genid/Yleaf) и визуализации по популяциям. 
На вход подаётся стандартный VCF-файл с генетическими данными, который скрипт последовательно подготавливает к анализу: извлекает только варианты Y-хромосомы, приводит их к единым координатам относительно референса hg19 и отфильтровывает низкокачественные вызовы. Поскольку данные часто получены с генотипирующих чипов, в которых отсутствует поле глубины прочтений, необходимое для расчёта статистической уверенности, отдельный модуль (pre_yleaf.py) автоматически формирует его на основе имеющихся генотипов. Это позволяет корректно адаптировать файл для запуска Yleaf.
После подготовки данных Yleaf сопоставляет выявленные маркеры с актуальной филогенетической базой и присваивает каждому образцу конкретную гаплогруппу с оценкой достоверности. Полученные результаты объединяются с информацией о популяционной принадлежности, после чего скрипт визуализации (yhaplo_visual.py) строит интерактивную сетку круговых диаграмм. Все этапы объединяются внутри yhaplo.sh

В директории запуска, помимо yhaplo.sh, pre_yleaf.py и yhaplo_visual.py, должны находиться:
1. groups.csv (sep=';') - таблица с колонками 'ID' и 'pop', где 'ID' - название образца, 'pop' - популяция, к которой образец относится;
2. hg19.no_alt.fa + .fai - проиндексированный hg19 референс;
3. cohort.vcf.gz + .csi - проиндексированный vcf файл с данными образцов.

Запуск: `./yhaplo.sh`

После выполнения в текущей директории появятся:
1. hg_prediction.hg - результат анализа y-гаплогрупп;
2. yhaplo_pop.html - интерактивная визуализация долей гаплогрупп по популяциям.
