# 2304

Скрипт для анализа гаплогрупп и визуализации по популяциям.

В директории запуска должны находиться:
1. groups.csv (sep=';') - таблица с колонками 'ID' и 'pop', где 'ID' - название образца, 'pop' - популяция, к которой образец относится;
2. hg19.no_alt.fa + .fai - проиндексированный hg19 референс;
3. cohort.vcf.gz + .csi - проиндексированный vcf файл с данными образцов.

Запуск: ./yhaplo.sh

После выполнения в текущей директории появятся:
1. hg_prediction.hg - результат анализа y-гаплогрупп;
2. populations_grid_final.html - интерактивная визуализация долей гаплогрупп по популяциям.
